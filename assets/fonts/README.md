# Font League Spartan

## Download Font
Download League Spartan dari Google Fonts:
https://fonts.google.com/specimen/League+Spartan

## Instruksi:
1. Download semua weights (Regular, Medium, SemiBold, Bold)
2. Extract file .ttf
3. Copy file-file berikut ke folder ini:
   - LeagueSpartan-Regular.ttf
   - LeagueSpartan-Medium.ttf
   - LeagueSpartan-SemiBold.ttf
   - LeagueSpartan-Bold.ttf

## Atau gunakan package Google Fonts (Lebih mudah):
```yaml
dependencies:
  google_fonts: ^6.1.0
```

Tidak perlu download manual jika menggunakan package.
